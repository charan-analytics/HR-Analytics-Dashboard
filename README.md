# HR-Analytics-Dashboard
Power BI HR Analytics Dashboard analyzing employee attrition, workforce metrics, and key HR KPIs using interactive visuals and insights.


## project overview
-An end-to-end HR Analytics Dashboard using Power BI and Excel that analyzes employee attrition, age, gender, department, salary, and experience. The project demonstrates data cleaning, KPI creation, and storytelling through interactive dashboards for effective HR decision-making.

## Tools used
-Power Bi

-Excel

## Key Kpi's
-Total employees
-Attrition Rate
-Average salary
-Average age
-Average Years

## Dashboard features
-Interactive slicers
-Attrition analysis
-kpis for quick insights

## Insights
-Highest attrition observed in laboratory technician
-Attrition is higher in between younger age groups.
